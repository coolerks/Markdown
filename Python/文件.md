# 文件

格式:```open(file,mode)```,两者均为字符串格式，与C相同

```f=open("D:\\1.txt","w",encoding="UTF-8")```

|  打开模式  | 说明                                                     |
| :--------: | -------------------------------------------------------- |
| ```'r' ``` | 以只读的方式打开文件                                     |
| ```'w' ``` | 以写入的方式打开文件，如果文件存在则覆盖                 |
| ```'x' ``` | 如果文件存在，则以此方式打开会提示错误（创建文件并打开） |
| ```'a' ``` | 以写入的模式打开，如果文件存在则在末尾追加               |
| ```'b' ``` | 以二进制打开文件                                         |
| ```'t' ``` | 以文本方式打开(默认)                                     |
| ```'+' ``` | 可读写模式                                               |
| ```'u' ``` | 使用通用换行符支持                                       |

|            文件对象            | 操作                                                         |
| :----------------------------: | ------------------------------------------------------------ |
|        ```f.close()```         | 关闭文件，Python有垃圾收集机制 不会发生内存泄露情况 可不关闭 |
|     ```f.read(size=-1)```      | 在文件中读取size个字符，未给定size或者size为负值时将读取所有字符，然后作为字符串返回 |
|       ```f.write(str)```       | 会将字符串```str```写入到文件                                |
|         ```f.tell()```         | 返回文件当前的指针位置                                       |
|     ```f.seek(len,from)```     | ```from```为```0```时代表起始位置，为```1```代表当前位置，为2代表末尾，移动```len```个位置 |
|       ```f.readline()```       | 读取一行                                                     |
|      ```tuple/list(f)```       | 将文件转为元组/列表                                          |
| ```f.writelines(list/tuple)``` | 将一个列表/元组传入并写入到文件                              |
