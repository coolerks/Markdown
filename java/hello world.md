# 你好，世界

代码

```java
public class helloworld {
    public static void main(String[] args)
    {
        System.out.println("Hello World");
    }
}
```

java中最小单位是类，类名```helloworld```应与文件名一致，一个java至少有一个类

java程序入口是```main```方法，格式为

```java
public static void main(String[] args){
    代码块
}
```
