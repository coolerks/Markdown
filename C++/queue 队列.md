# queue队列

先进先出，first in first out

队头只能出数据，队尾只能入队

只有队头和队尾才能被外界使用，因此队列不允许有遍历行为

| 操作                  | 说明         |
| --------------------- | ------------ |
| ```q.size()```        | 队列大小     |
| ```q.empty()```       | 队列是否为空 |
| ```queue<type>q```    | 创建         |
| ```q.push(element)``` | 入队         |
| ```q.pop()```         | 出队         |
| ```q.front()```       | 返回队头元素 |
| ```q.back()```        | 返回队尾元素 |
