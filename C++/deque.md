# deque容器

#####功能

又称**双端数组**，可对**尾端**和**头端**数据进行添加和删除

##### 与vector的区别

vector在头部插入时效率比较低

deque的头部插入效率高，但访问时的效率较低

| 写法               | 功能                     |
| ------------------ | ------------------------ |
| ```deque<type>d``` | 初始化d为type类型的deque |
| ```其他 操作与vector一致``` |                          |
