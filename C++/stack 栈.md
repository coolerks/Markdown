# 栈（stack）

先进后出

后进先出

**栈**不允许有遍历的行为，只有栈顶的元素才可以被访问

栈可以判断是否为**空**以及**大小**

## 构造

| 方式                   | 说明     |
| ---------------------- | -------- |
| ```stack<type> 变量``` | 默认构造 |

## 数据存取

| 方式                  | 说明                 |
| --------------------- | -------------------- |
| ```st.push(value) ``` | 入栈                 |
| ```st.pop()```        | 出栈，不会返回任何值 |
| ```st.top()```        | 返回栈顶元素         |

## 大小

| 方式             | 说明         |
| ---------------- | ------------ |
| ```st.size()```  | 返回栈的大小 |
| ```st.empty()``` | 栈是否为空   |
